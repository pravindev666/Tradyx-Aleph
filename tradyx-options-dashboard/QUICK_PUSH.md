# Quick GitHub Push Commands

## Fastest Way to Push (Recommended)

```bash
# 1. Navigate to project
cd "C:\Users\hp\Desktop\Desktop Placed\Tradyx\tradyx-options-dashboard"

# 2. Initialize (if first time)
git init

# 3. Add remote (replace YOUR_USERNAME and REPO_NAME)
git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git

# 4. Add files (respects .gitignore - excludes node_modules, data, etc.)
git add .

# 5. Commit
git commit -m "Initial commit: Tradyx Options Dashboard"

# 6. Push
git push -u origin main
```

## If You Get "Large File" Error

### Option A: Use Git LFS (for files > 100MB)
```bash
# Install Git LFS first: https://git-lfs.github.com/
git lfs install
git lfs track "*.json"
git add .gitattributes
git add .
git commit -m "Add LFS tracking"
git push -u origin main
```

### Option B: Exclude Large Files
```bash
# Remove large file from git
git rm --cached large-file.json

# Add to .gitignore
echo "large-file.json" >> .gitignore

# Commit and push
git add .gitignore
git commit -m "Exclude large files"
git push
```

## Check What Will Be Pushed

```bash
# See what files are staged
git status

# See file sizes
git ls-files | ForEach-Object { Get-Item $_ | Select-Object Name, @{Name="Size(MB)";Expression={[math]::Round($_.Length/1MB,2)}} }
```
